## How Licensing Works

```
pip install neural-memory
nmem pro activate <license-key>
```

1. **Purchase** via Polar.sh or Sepay → receive license key via email
2. **Install** the Pro package
3. **Activate** with your license key
4. Pro features unlock immediately — InfinityDB auto-initializes on next startup

**License is per-machine for Pro, per-seat for Team.**

Offline activation available for air-gapped environments (Team plan).

## License Verification

Pro validates the license locally with a lightweight check:

- **Online**: Periodic heartbeat to license server (1x per 7 days)
- **Offline grace**: Works without internet for 30 days after last verification
- **No telemetry**: Zero usage data sent. Only license key + machine ID for validation

## FAQ

**Can I try Pro before buying?**
Not yet — but the 30-day refund policy is effectively a free trial. Install, test, refund if not satisfied.

**What happens when my subscription expires?**
Your agent falls back to free SQLite. InfinityDB files stay on disk. Resubscribe anytime to resume.

**Can I use one license on multiple machines?**
Pro license is per-machine. Use Team plan for multi-machine or multi-developer setups.

**Do you offer student/OSS discounts?**
Yes. Active open-source contributors and students get 50% off Pro. Email us with proof.

**Is there a free tier for cloud sync?**
Yes — free plan includes 100 neurons synced via Cloudflare hub. Pro unlocks unlimited sync.

**Can I pay via crypto?**
Not currently. If there's demand, we'll add USDT/USDC via a future payment provider.
